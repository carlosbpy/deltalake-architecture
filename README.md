# POC (Proof of Concept) Delta Lake
- Create Virtual Environment
    ```sh
    python -m venv venv
    ```
- Load Virtual Environment
    ```sh
    source venv/bin/activate
    ```
- Install Dependencies
    ```sh
    brew install openjdk@11
    pip install -r requirements.txt
    ```
- Execute Main Job
    ```sh
    python main.py
    ```
